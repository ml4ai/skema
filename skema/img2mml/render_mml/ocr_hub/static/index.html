OCR Hub

  • Mathpix OCR Tweak-er
    A webapp to individually visualize the Mathpix annotations and make any manual corrections required. Both the LaTeX code and the MathML code generated by Mathpix can be manually corrected to match the original image here.
  • OCR Verify
    A webapp to visualize the OCR results of our own model and compare the output MathML to the original image.
