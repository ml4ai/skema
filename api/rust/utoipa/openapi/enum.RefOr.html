Enum utoipa::openapi::RefOr

source · 
pub enum RefOr<T> {
    Ref(Ref),
    T(T),
}
Expand description

A Ref or some other type T.

Typically used in combination with Components and is an union type between Ref and any other given type such as Schema or Response.
